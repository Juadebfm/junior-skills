# TaskFlow

TaskFlow is a simple task management web app for beginners who want to practice building CRUD features and basic productivity UI.

## Main idea

Users can create tasks, mark them complete, group them by status, and track what needs attention today.

## Main screens

- Landing page
- Sign up
- Sign in
- Dashboard
- Task details or edit modal

## Main features

- Create a task
- Edit a task
- Mark a task complete
- Filter by status
- Show due today and overdue items

## Good beginner MVP

If the full app feels too big, start with:

1. Landing page
2. Dashboard with fake task data
3. Create task form

Then add:

1. Edit task flow
2. Status filters
3. Real saved data
