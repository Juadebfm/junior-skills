# Example Prompts For TaskFlow

These are simple examples a beginner could use with this skills pack.

## Understand the project

```text
Use understand-brief on this TaskFlow brief and tell me the simplest MVP I should build first.
```

## Grow the project safely

```text
Use grow-project and help me add labels, due dates, and simple analytics without making the app too complex.
```

## Plan frontend and backend

```text
Use plan-stack and plan the frontend pages, backend routes, and task data model for this TaskFlow MVP.
```

## Plan the design system

```text
Use design-system-plan and help me define a clean, simple UI system for TaskFlow.
```

## Plan the backend system

```text
Use backend-system-design and help me design the task APIs, task data model, auth approach, and safe backend structure for TaskFlow.
```

## Build one part

```text
Use build-small and help me build the create task form first.
```

## Decide what to test

```text
Use test-small and tell me what I should test for task creation, task editing, and filter behavior.
```

## Review the result

```text
Use check-my-work and review my dashboard and task form for the most important issues.
```

## Check security basics

```text
Use security-check and review my form validation and API writes for common mistakes.
```

## Check coding standards

```text
Use standards-check and tell me whether my folders, naming, and repeated patterns still make sense.
```
