# Example Prompts For CodeTrack

These are simple examples a beginner could use with this skills pack.

## Understand the project

```text
Use understand-brief on this CodeTrack brief and tell me the easiest version I should build first.
```

## Grow the project safely

```text
Use grow-project and help me add goals, skill tracking, and charts without making the app too complex for me right now.
```

## Plan frontend and backend

```text
Use plan-stack and plan the frontend pages, backend routes, and data models for this CodeTrack MVP.
```

## Plan the design system

```text
Use design-system-plan and help me plan the colors, type scale, spacing, cards, buttons, form fields, and responsive rules for CodeTrack.
```

## Plan the backend system

```text
Use backend-system-design and help me design the APIs, data models, auth flow, and future-safe backend structure for CodeTrack.
```

## Build one part

```text
Use build-small and help me build the landing page first.
```

## Decide what to test

```text
Use test-small and tell me what I should test on the sign-up form and dashboard.
```

## Review the result

```text
Use check-my-work and review my dashboard page for the most important issues.
```

## Check security basics

```text
Use security-check and review my auth flow and form handling for common security mistakes.
```

## Check coding standards

```text
Use standards-check and tell me whether my project structure and naming are staying consistent.
```
