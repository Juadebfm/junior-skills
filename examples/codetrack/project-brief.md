# CodeTrack

CodeTrack is a web app for beginner developers who want to build a daily coding habit.

## Main idea

Users log what they studied or built each day and watch a streak grow over time.

## Main screens

- Landing page
- Sign up
- Sign in
- Dashboard

## Main features

- Daily log entries
- Streak tracking
- Goal progress
- Skills or topic tracking

## Good beginner MVP

If the full app feels too big, start with:

1. Landing page
2. Sign up page
3. Dashboard with fake data

Then add:

1. Real forms
2. Real saved log entries
3. Goal progress
