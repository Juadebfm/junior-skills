# Security Policy

If you discover a security issue in this repository, please do not report it in a public GitHub issue.

Instead:

1. Contact the maintainer privately through GitHub.
2. Include enough detail to reproduce and understand the issue.
3. Share any impact you are aware of.

Please avoid posting:

- secrets
- tokens
- private credentials
- private user data

Public issues are fine for normal bugs, documentation problems, and feature requests, but not for security disclosures.
